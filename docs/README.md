# plentyshop-pwa-docs

## Starting the app

1. Open the repository in your command line interface.
2. Run `npm install` to install all dependencies.
3. Run `npm run docs:dev` to start the development server. The app will be served with hot reload at [localhost:5173](http://localhost:5173/).

## What's published

Directories and files prefixed with an underscore are excluded from publication at https://pwa-docs.plentyone.com/

While this is a public repository and all documentation is technically available as a result, content should be excluded from publication under the following circumstances:

- Technical details are still subject to change
- The content is work-in-progress
- The content is only relevant for the core development team
